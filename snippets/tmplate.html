AWD Resources

Educational website

W3School is one of the most used educational websites for learning web development. It contains may simple to understand examples and has a large number of web templates. w3school contains all of the material that we will use on this course.

Editor

The choice of an editor or integrated development environment is up to you. The editor I recommend is ATOM. ATOM is a highly customisable editor with many community sourced packages. The atom-live-server package allows local web development without having to install your own local web server.

Web publishing

We will use the version control hosting web site GitHub to publish our webpages. See tutorial 1 for a guide on how to set up your project web page. GitHub is also responsible for the ATOM editor so is tightly integrated with it.
