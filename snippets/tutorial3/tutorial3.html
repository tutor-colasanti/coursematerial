CSS

Controlling how your web page looks

Elements

                            
    body {
        background-color: #0D47A1;
    }

    h1 {
        color: #E3F2FD;
    }

Place this code in your main.css file.

Here we are using the two elements 'body' and 'h1' and setting their color.

See w3school color for other ways to do color
