Responsiveness

A very good example of a responsive web page is Chrome box unboxed

See how far you can mirror its design. Initially it looks quite simple, and you should be able to mimic the column and card design that collapse to a single column when viewed on smart phone.

The layout of the cards is more complex. Note border radius and the transition plus shadow when mouse over. For a more challenging task try and copy the layout of the image and the text of the card. Click on the card and see the full image and see how the full image looks? How will you get the truncated image in the card?
