Advanced web design

Responsive web design

This course in advanced web design will introduce you to the practice of responsive web design. Responsive web design is the method of developing web pages so that they will be optimised for a range of viewing platforms. By adopting RWD your page will look great on a desktop, laptop, tablet or smartphone.

Web design libraries

In creating responsive web designs you will learn advanced HTML, CSS and how to incorporate two of the most highly used web design libraries jQuery and Boostrap 4. We will use both these libraries to create single page web applications that interact with a simple database via PHP using a REST interface.

Model View Controller

You will also be introduced to the design pattern Model View Controller(MVC). This will help you organise your pages and your website for optimum performace and utility
